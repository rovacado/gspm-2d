* GSPM-2D
This is a super simple template for the [[https://pypi.org/project/gspm/][gspm]] CLI tool for Godot.

* How to Use
first clone the repo
#+begin_src shell
git clone https://github.com/rovacado/gspm-2d.git
#+end_src
CD into the directory and edit project.yml as needed, then run the installation command
#+begin_src shell
gspm install
#+end_src
finally, run the edit command to launch Godot straight into the project
#+begin_src shell
gspm edit
#+end_src
